# **GithubCard**
Webapp that creates a card base on a Github profile, using the Github REST API.

## **Run Locally**

Clone the project

```bash
git clone https://github.com/risixdzn/githubcard.git
```

Go to the project directory

```bash
cd esfihasdash
```

Install dependencies

```bash
npm install | yarn
```

Start the server

```bash
npm run dev | yarn run dev
```
